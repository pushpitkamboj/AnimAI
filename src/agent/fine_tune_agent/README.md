# Legacy Fine-Tune Graphs

This directory contains deprecated experimental graph implementations that are
kept as reference material only.

They are not used by the active production flow:

`src/api/main.py` -> `src/agent/graph.py` -> `src/agent/execute_code.py` -> `manim-worker/app.py`

The files remain in the repository because they may still be useful for future
research, prompt comparison, or selective recovery of ideas. They should be
treated as legacy code, not maintained production paths.
